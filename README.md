# ATM-Project
by Python and SQL

## 一、	目的
使用Python與SQL 模擬一個簡單的ATM系統，具備的功能:管理員登入/登出、使用者開戶、登入、修改密碼、查詢、存取款、轉帳等功能。藉由SQLite 創建一個database，將所有使用者紀錄其中，因此來學習ATM的運作。

## 二、	功能
### 1.	Admin：
  * 開戶
  * 查詢帳戶餘額
  * 提款
  * 存款
  * 轉賬
  * 修改密碼
  * 鎖定帳戶
  * 解鎖帳戶
  * 補辦卡片
  *	銷除帳戶
 
###  2.	Person
  * 開戶
  * 查詢帳戶餘額
  * 提款
  * 存款
  * 轉賬
  * 修改密碼
  * 開辦卡片
 
  
## 三、	使用技術
1.	SQLite
  i.	CREATE
  ii.	INSERT
  iii.	SELECT
  iv.	DELETE
  v.	DROP
2.	Random
